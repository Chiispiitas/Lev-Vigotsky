Paper Mode Q-cards
===================

The printable student cards use universal 6x6 Q-codes.
They are not session-specific. Print them once and reuse the same cards for any Paper Mode game.

Students answer by rotating the card so the chosen letter is above the 6x6 code:
A = normal orientation
B = rotated so B is at the top
C = upside down
D = rotated so D is at the top

The phone scanner session QR is still session-specific and points to:
https://chiispiitas.github.io/Lev-Vigotsky/Quiz/scanner.html

Camera scanning requires HTTPS or localhost. Opening index.html through file:// may block camera access.
